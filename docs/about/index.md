# About Junkmesh

Junkmesh turns any x86 machine — especially the retired laptop in your
cupboard — into a node of a fully decentralised storage cloud. Yggdrasil
provides an encrypted global mesh with no coordinator; Garage replicates
your data across the nodes; a tiny exporter tells your monitoring what's
going on. Everything runs as plain Alpine Linux services from one
installable ISO. No containers, no cloud account, no one in charge.
